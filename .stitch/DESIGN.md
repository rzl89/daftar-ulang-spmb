# Design System: SPMB SMKN 5 Kota Serang

## 1. Color Palette
- **Primary (Navy Blue):** `#1A237E` (Deep education/authority tone)
- **Secondary (Gold):** `#F9A825` (Premium accents, highlights, call-to-actions)
- **Background:** `#F8FAFC` (Slate-light background for pages), `#FFFFFF` (Card background)
- **Text:** `#0F172A` (Dark slate for headings), `#475569` (Slate for body)

## 2. Typography
- **Primary Font:** `Inter` or `Poppins`
- **Headings:** Bold, Navy Blue, clear hierarchy
- **Buttons:** Semi-bold, readable

## 3. Component Style
- **Corners:** Rounded corners with `16px` (or `rounded-2xl` in Tailwind)
- **Shadows:** Subtle shadows (`shadow-sm` or `shadow-md`) for container cards
- **Buttons:** Interactive hover animations, pulse effects on primary gold buttons

## 4. UI Elements & Icons
- Use modern SVG icons (Heroicons/Lucide style)
- Step indicators for progress forms
- Glassmorphism overlays on gradients
